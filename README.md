# vti-crc-projekt
Python program for finding valid crc generating polynomials and encoding binary strings

Dependencies zabaleny pomocí venv

Aktivace venv skrze PowerShell:
```
./Scripts/activate
```

Python verze: 
    3.11.0

Použité knihovny:
    CustomTKinter 5.0.3

Instalace knihoven:
```
pip install customtkinter 
```
